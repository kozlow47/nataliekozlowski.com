CodeGuard Homepage Website redesign

This project was extremely fun for me because I managed it from start to finish. I created the design and all custom graphics using Photoshop and Illustrator. For development I used Middleman, a ruby based static site generator. I designed and built three custom animations to help illustrate what our application does. I also utilized multiple JavaScript waypoint and parallax effects to keep visitors interested throughout the experience. Most of the development was done in HTML, CSS and JavaScript. The website is completely responsive and mobile-optimized. View this project and the custom animations live at the CodeGuard website!
